Introduction

Material Design Doc demonstrates two ideas: 1. How to use the Angular Material UI Framework, and 2. How to create a Style Guide.


What's a UI Framework?

UI Frameworks are a collection of modular components that can be added on top of an app. Angular Material implements Google's Material Design system with the Javascript MVC AngularJS.

What's a Style Guide?
